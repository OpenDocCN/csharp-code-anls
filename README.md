## 下载

### Docker

```
docker pull apachecn0/csharp-code-anls
docker run -tid -p <port>:80 apachecn0/csharp-code-anls
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g csharp-code-anls
csharp-code-anls <port>
# 访问 http://localhost:{port} 查看文档
```
